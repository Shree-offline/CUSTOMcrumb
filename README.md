🍰 CUSTOMcrumb: The Dynamic Cake Architecture Engine

CUSTOMcrumb is a professional-grade web application designed to bridge the gap between complex bakery logistics and user-friendly customer experiences. It automates the "price-per-slice" headache by using a dynamic calculation engine that scales in real-time as users design their dream cake.

✨ Key Features

The application provides a granular customization suite, allowing users to define:
Dynamic Themes: Specialized pricing for Weddings, Birthdays, or "Prank" cakes.
Structural Scaling: Price adjustments based on the number of layers (1 to 5+).
Children’s Section: A dedicated module for licensed or complex designs (Barbie, Robots, Transformers).
Palette Control: Price-sensitive color selections (Standard vs. Deep Pigment).
Gourmet Flavors: Real-time premiums for exotic bases like Cookies n' Cream or Rich Chocolate.
